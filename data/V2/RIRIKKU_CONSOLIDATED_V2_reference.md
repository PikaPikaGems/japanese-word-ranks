# RIRIKKU_CONSOLIDATED_V2.csv — Reference

## Overview

`data/ALL/RIRIKKU_CONSOLIDATED_V2.csv` is the V2 successor to `RIRIKKU_CONSOLIDATED.csv`.
It uses the same word list, source shortlist, and ranking formula as V1, but replaces the
bidirectional kana/kanji fallback with a **gated merging algorithm** that prevents
false-positive homonym merges. See `notes/RIRIKKU_CONSOLIDATED_MERGING_ALGO_V2.md` for
the full algorithm design.

Generated by: `data/ALL/___data_generation/make_ririkku_v2.py`

## Columns

| Column                 | Type   | Description                                                                                                                         |
| ---------------------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| `word`                 | string | The word as it appears in the source datasets (kanji/kana/mixed)                                                                    |

### RIRIKKU_TIER buckets

| Tier          | Rank range    | Description                                                |
| ------------- | ------------- | ---------------------------------------------------------- |
| B: `BASIC`    | 1–1,800       | Core everyday vocabulary                                   |
| C: `COMMON`   | 1,801–5,000   | Common vocabulary                                          |
| F: `FLUENT`   | 5,001–12,000  | Fluency-level vocabulary                                   |
| A: `ADVANCED` | 12,001–25,000 | Advanced vocabulary                                        |
| U: `UNRANKED` | —             | Fewer than 3 sources, or all source ranks outside 1–25,000 |

Tier is assigned by **majority vote**: each source's rank is bucketed, and the tier with
the most votes wins. Ties go to the higher (better) tier.

## Differences from V1 (RIRIKKU_CONSOLIDATED.csv)

### New column: RIRIKKU_TIER

V1 does not have a `RIRIKKU_TIER` column. V2 adds it between `RIRIKKU_RANK` and the
source columns.

### Gated merging algorithm

The core difference. V1 uses unrestricted bidirectional kana/kanji fallback: when a word
is not found directly in a source, it tries the word's kana reading (kanji→kana) and all
kanji forms sharing that reading (kana→kanji). This causes **false-positive merges** where
obscure homonyms inherit the rank of common words.

**Example (V1 bug):** 埜 (archaic kanji for "field", reads as の) inherits the rank of
the particle の across all sources, placing it near rank #1. In V2, 埜 is correctly
`RIRIKKU_RANK = -1` (UNRANKED).

V2 gates the fallback with three mechanisms:

| Mechanism        | Purpose                                                                                                                                                                      |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MUST_MERGE`     | Whitelisted pairs (UniDic lemma ↔ surface form) that always merge regardless of other gates. E.g. 為る↔する, 貰う↔もらう.                                                    |
| `MUST_NOT_MERGE` | Blacklisted pairs that never merge, even if the automated gate would allow it. E.g. 帰る↔かえる (return home ≠ change/alter).                                                |
| Automated gate   | Only the single highest-frequency kanji form for a given reading (per JPDB) is merge-eligible, and only if it appears directly in ≥2 recommended sources within the top 30k. |

### Statistical impact

| Metric            | V1             | V2             | Diff     |
| ----------------- | -------------- | -------------- | -------- |
| Total words       | 88,716         | 88,716         | —        |
| Ranked (≥3 src)   | 42,730         | 41,947         | **−783** |
| Unranked (<3 src) | 45,986         | 46,769         | +783     |
| BASIC             | 3,025 (3.4%)   | 2,802 (3.2%)   | −223     |
| COMMON            | 4,347 (4.9%)   | 4,257 (4.8%)   | −90      |
| FLUENT            | 10,723 (12.1%) | 10,632 (12.0%) | −91      |
| ADVANCED          | 24,635 (27.8%) | 24,256 (27.3%) | −379     |
| UNRANKED          | 45,986 (51.8%) | 46,769 (52.7%) | +783     |

783 words that V1 ranked are now correctly unranked in V2. These were words that only
appeared to qualify because false-positive merges gave them ranks they didn't earn
independently. The tier drops (−223 BASIC, −379 ADVANCED etc.) reflect the same
correction: words that were over-promoted due to inherited homonym ranks are now
assigned to lower tiers or UNRANKED.
